# Changing from M.S. to Ph.D.

An M.S. student can change to the Ph.D. program providing the entrance examination requirement has been met.

Per [Graduate School policy]({{ URLAcadPolicies }}), a student originally admitted to the M.S. program must formally reapply for Ph.D. admission and receive approval from the Admissions Subcommittee, even if enrollment at UWM has continued without interruption.
